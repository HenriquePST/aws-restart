Porque Usar um banco de dados gerenciado na AWS ?

AWS gerencia o banco de dados automaticamente 
 Provisionamento rápido 
 High availability 
 Escalonamento Vertical e Horizontal 
 Backups/Restores Automatizados 
 Updates/Upgrades automatizados 
 Monitoramento/Alertas Integrados 

 Bancos não Gerenciados precisa fazer deploy do DB em uma EC2, mas você
precisaria gerencia todos os itens acima.
